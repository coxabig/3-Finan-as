# Política de Privacidade - 3%

**Última atualização: 28 de Abril de 2026**

A sua privacidade é nossa prioridade. Esta Política de Privacidade descreve como o app **3%** coleta, utiliza e protege suas informações.

## 1. Coleta de Dados
Coletamos os seguintes dados para funcionamento do serviço:
- **Informações de Perfil:** Nome, e-mail e foto (via Login Social ou cadastro direto).
- **Dados Financeiros:** Receitas, despesas, faturas importadas e metas. Estes dados são armazenados de forma criptografada no Google Firebase.
- **Analytics:** Coletamos dados anônimos de uso (páginas visitadas, tempo de uso) via Google Analytics for Firebase para melhoria do app.

## 2. Uso dos Dados
Os dados coletados são utilizados exclusivamente para:
- Calcular a proporcionalidade de despesas entre os membros do casal.
- Gerar dashboards financeiros.
- Sincronizar dados entre os dispositivos do mesmo casal.
- Enviar notificações sobre vencimentos e metas.

## 3. Login Social
Ao utilizar o login via Google ou Apple, recebemos apenas o seu ID de usuário, nome e e-mail. Não temos acesso às suas senhas ou outros dados privados dessas plataformas.

## 4. Compartilhamento de Dados
Não vendemos nem compartilhamos seus dados financeiros com terceiros. Seus dados são compartilhados apenas com o parceiro(a) que você autorizar o vínculo no app.

## 5. Exclusão de Dados
Você pode solicitar a exclusão total de sua conta e dados a qualquer momento através das configurações do perfil no aplicativo.

## 6. Permissões
- **Notificações:** Para avisar sobre gastos e metas.
- **Arquivos/PDF:** Para importação de faturas de cartão de crédito.

Ao utilizar o 3%, você concorda com os termos desta política.
